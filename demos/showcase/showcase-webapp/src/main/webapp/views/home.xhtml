Liferay Faces Showcase

Liferay Faces is an umbrella project that provides support for the JavaServerTM Faces (JSF) standard within Liferay Portal. This showcase application demonstrates usage of Facelet component tags provided by the following sub-projects:

Liferay Faces Alloy

The Liferay Faces Alloy sub-project provides a suite of JSF UI components that utilize Alloy UI.

The Facelet component tags provided by Liferay Faces Alloy are prefixed with the aui: namespace.

Liferay Faces Bridge

The Liferay Faces Bridge is a portlet bridge for JSF which provides the ability to deploy JSF web applications as portlets within Liferay Portal.

The Facelet component tags provided by Liferay Faces Bridge are prefixed with the bridge: and portlet: namespaces.

Liferay Faces Portal

The Liferay Faces Portal sub-project provides JSF utilities, EL keywords, and a suite of JSF UI components that are Facelet equivalents of the JSP tag libraries provided by Liferay Portal.

The Facelet component tags provided by Liferay Faces Portal are prefixed with the liferay-ui:, liferay-util, and liferay-security: namespaces.
