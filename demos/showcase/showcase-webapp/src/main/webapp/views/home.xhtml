Liferay Faces is an umbrella project that provides support for the JavaServerTM Faces (JSF) standard within Liferay Portal. This showcase application demonstrates usage of Facelet component tags provided by the following sub-projects:

The Liferay Faces Alloy sub-project includes a suite of JSF UI components that utilize Alloy UI.

The Facelet component tags are prefixed with the alloy: namespace and are compatible with webapp and portlet environments.

The Liferay Faces Bridge sub-project includes a JSR 329 certified portlet bridge which provides the ability to deploy JSF 1.x/2.x web applications as portlets within JSR 286 (Portlet 2.0) compliant portal servers such as Liferay Portal.

The Facelet component tags are prefixed with the bridge: and portlet: namespaces and are only compatible with portlet environments.

The Liferay Faces Portal sub-project includes JSF utilities, EL keywords, and a suite of JSF UI components for use with Liferay Portal.

The Facelet component tags are prefixed with the portal: namespace are only compatible with Liferay portlet environments.
